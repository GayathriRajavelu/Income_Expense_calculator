# Income and Expense Calculator

A simple web application to track personal income and expenses.

## Features
- Add and manage income entries
- Add and manage expense entries
- Calculate total balance
- View transaction history
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
1. Clone the repository
2. Open `index.html` in your browser

## Usage
1. Enter income/expense amount and description
2. Click Add button to record transaction 
3. View running balance and transaction history
4. Delete entries as needed
